    • SINTAXIS CSS

    Un conjunto de reglas CSS consta de un selector y un bloque de declaración


      Los puntos selector para el elemento HTML que desea estilo.
      El bloque de declaración contiene una o más declaraciones separadas por punto y coma.
      Cada declaración incluye un nombre de propiedad CSS y un valor, separados por dos puntos.
      Una declaración CSS siempre termina con un punto y coma, y los bloques de declaraciones están rodeados por llaves.

    Selectores básicos

    Selector de tipo

    Selecciona todos los elementos que coinciden con el nombre del elemento especificado.
    Sintaxis: eltname

    Selector de clase

    Selecciona todos los elementos que tienen el atributo de class especificado.
    Sintaxis: .classname

    Selector de ID

    Selecciona un elemento basándose en el valor de su atributo id. Solo puede haber un elemento con un determinado ID dentro de un documento.
    Sintaxis: #idname

    Selector universal

    Selecciona todos los elementos. Opcionalmente, puede estar restringido a un espacio de nombre específico o a todos los espacios de nombres. Sintaxis: * ns|* *|*
    Ejemplo: * se aplicará a todos los elementos del documento.
    Selector de atributo

    Selecciona elementos basándose en el valor de un determinado atributo.
    Sintaxis: [attr] [attr=value] [attr~=value] [attr|=value] [attr^=value] [attr$=value] [attr*=value]
